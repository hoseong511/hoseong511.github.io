리포지토리를 새로 만들었을때
```
git init
git add . # 소스 코드를 작업하는 영역에서 이루어진 작업 내용을 Git의 스테이징 영역으로 이동시킨다.
git commit -m "" # 로컬 Git의 리포지토리에 반영
git branch -M main
git remote add origin https://github.com/xxxx/xxxx.git
git push -u origin main # 원격 Git의 리포지토리에 반영
```


다른 pc에서 깃허브 작업을 할때
```
git clone
git add .
git commit -m ""
git push
```
